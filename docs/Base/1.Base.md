# 1. Desenho de Software

Este modulo apresenta os artefatos produzidos na fase Base do projeto Battle Class, organizados nos seguintes topicos:

- [1.2. Artefato Generalista](Base/1.2.ArtefatoGeneralista.md) - Artefato independente de metodologia elaborado pela equipe.
- [1.3. Modelagem BPMN](Base/1.3.ModelagemBPMN.md) - Modelagem na notacao BPMN orientada pela abordagem metodologica adotada.
- [1.4. Participacoes](Base/1.4.ParticipacoesBase.md) - Quadro de participacoes de cada membro na entrega.
- [1.5. Iniciativas Extras](Base/1.5.IniciativasExtras.md) - Iniciativas extras realizadas pela equipe.

> **Para cada foco**: Rastreabilidade & Elos com Outros Artefatos, Senso Critico, Referencias, Versionamentos & Participacoes e Metodologia.
